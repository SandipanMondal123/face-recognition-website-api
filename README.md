# face-recognition-website-api

cd into the project
npm install
npm start
This is the back-end component which utilized **Node.js, Express, Knex.js, PostgreSQL (database)** and deployed on **Heroku**. If you want to see the backend, check out this link https://github.com/SandipanMondal123/face-recognition-website-api .

Demo: https://face-recognition-website-sm.herokuapp.com/
